# To-Do App

A simple, single-page to-do app with persistence. No build step required.

## Features

- **Add tasks** — Type and press Add or Enter
- **Mark complete** — Click the circle next to a task
- **Delete** — Click the × on a task
- **Filter** — All, Active, or Done
- **Clear completed** — Remove all done tasks at once
- **Persist** — Tasks are saved in your browser (localStorage)

## Run

Open `index.html` in a browser (double-click the file or use “Open with” and choose your browser). You can also use a local server:

```bash
cd todo-app
python3 -m http.server 8000
```

Then visit http://localhost:8000
